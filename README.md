# bonosweb
SaaS Multi-tenant (Laravel 12 + Livewire 4) y App Móvil (Ionic 8 + Angular 20) para gestión y firma electrónica de recibos de sueldo/liquidaciones. Procesamiento OCR de PDFs, firmado criptográfico con certificados digitales, pasarela de facturación central con control de inflación y suspensión automática.
